## 📌 Project Description

Smart Task Manager is a responsive and user-friendly task management web application built using **HTML, CSS, and JavaScript**.

This application helps users efficiently manage daily tasks by allowing them to:

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑 Delete tasks
- 💾 Save tasks using localStorage
- 📱 Use the app across different screen sizes

The project is beginner-friendly and demonstrates core frontend development concepts such as DOM manipulation, event handling, responsive design, and browser storage.
